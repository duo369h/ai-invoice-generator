Date: 2026-07-01
Scope: Product narrative, workflow section, preview captions. Copy pack, no code changes.

## Narrative Principle

Corvioz should describe a structured client workflow system, not a billing funnel.

The product story should move through:

```text
Request -> Quote -> Proposal -> Client Review -> Delivery Record
```

It should not move through:

```text
Quote -> Invoice -> Payment -> Revenue
```
