# Copy Changes

## Landing Page

Changed the primary product narrative from financial or revenue framing to a structured client workflow narrative.

Key replacements:

- Hero identity changed to `Freelancer Workflow System`.
- Hero headline changed to `Run every client workflow with structure.`
- Supporting copy now frames Corvioz around quotes, proposals, client documents, and delivery records.
- Product preview labels changed from payment/revenue states to workflow states:
  - `Client delivery flow`
  - `Quote to client approval`
  - `Client document delivered`
  - `Approval`
  - `Delivery`
  - `Workspace timeline updated`

## Product Narrative

Updated workflow steps to approved safe positioning:

- Capture Request
- Create Quote
- Prepare Proposal
- Manage Delivery

The product is now described as a workspace for organizing client work, not as a system for moving or collecting money.

## Pricing

Preserved four-plan structure:

- Free
- Starter
- Pro
- Studio

Updated plan descriptions and features to focus on workflow maturity, document volume, client operations, and delivery structure.

Updated CTAs:

- Free: `Start Free`
- Starter: `Choose Starter`
- Pro: `Choose Pro`
- Studio: `Choose Studio`

Paddle wording is constrained to secure checkout provider language.

## Dashboard And Product Preview

Replaced high-risk status terms with safe workflow terms:

- `Paid` -> `Completed`
- `Revenue` -> `Delivery`
- `Payment link delivered` -> `Client document delivered`
- `Transfer recorded` -> `Workflow updated`
- `Workspace balance updated` -> `Workspace timeline updated`

Internal status values were not changed when they support existing application logic.

## Modals And Upgrade Prompts

Rewrote upgrade and restriction copy to avoid ROI, financial promise, or payment processor positioning.

Safe patterns used:

- `Upgrade Workspace`
- `Unlock Clean Export`
- `Choose Starter`
- `Choose Pro`
- `Choose Studio`
- `Secure checkout`

## Footer, Legal, Trust, And Security

Updated footer and trust copy to align with the hero identity and pricing page.

Paddle appears only as a secure checkout provider for paid plans. Corvioz is not described as the checkout provider, payment processor, billing platform, or financial infrastructure.
